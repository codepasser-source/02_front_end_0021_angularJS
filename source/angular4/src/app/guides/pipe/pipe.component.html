Pipe Guide

        # Angular 4 中 Pipe(管道) 与 Angular 1.x 中的 filter(过滤器) 的作用的是一样的。
        # 它们都是用来对输入的数据进行处理，如大小写转换、数值和日期格式化等。

        # Angular 内置管道及分类
        String -> String
          UpperCasePipe
          LowerCasePipe
          TitleCasePipe
        Number -> String
          DecimalPipe
          PercentPipe
          CurrencyPipe
        Object -> String
          JsonPipe
          DatePipe
        Tools
          SlicePipe
          AsyncPipe
          I18nPluralPipe
          I18nSelectPipe

Section 自定义Pipe

自定义Pipe Guide

        # 自定义管道的步骤：

        - 使用 @Pipe 装饰器定义 Pipe 的 metadata 信息，如 Pipe 的名称 - 即 name 属性
        - 实现 PipeTransform 接口中定义的 transform 方法

        - 创建共同指令模块
        ng g m pipes
        - 创建指令
        ng g p pipes/ellipsis --spec false --export true

自定义Pipe Sample


        
{{ 'semlinker' | ellipsis:{digit:5,symbol:'...'} }}


        
{{ 'semlinker' | ellipsis:{digit:5,symbol:'...'} }}
