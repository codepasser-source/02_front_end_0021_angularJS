Section Directive

指令概念 Guide

        # Angular 的指令分为三种：
        - 组件(Component directive)：用于构建UI组件，继承于 Directive 类.
        - 属性指令(Attribute directive)：用于改变组件的外观或行为.
        - 结构指令(Structural directive)：用于动态添加或删除 DOM 元素来改变 DOM 布局.

Section 内置属性指令

ngStyle Guide

        ngStyle指令： 用于设定给定 DOM 元素的 style 属性
        # 绑定常亮
        <div [ngStyle]="{{'{'}}'background-color': 'green'{{'}'}}">...</div>
        # 绑定表达式
        <div [ngStyle]="{{'{'}}'background-color': condition === true ? 'red' : 'green'{{'}'}}">...</div>

ngClass Guide

        ngClass指令：用于动态的设定 DOM 元素的 CSS class
        # 绑定常亮
        <div [ngClass]="{{'{'}}'alert-info': true{{'}'}}">...</div>
        # 绑定表达式
        <div [ngClass]="{{'{'}}'alert-success': condition === true{{'}'}}">...</div>

Section 内置结构指令

ngIf Guide

        # 该指令用于根据表达式的值，动态控制模板内容的显示与隐藏。
        # 它与 AngularJS 1.x 中的 ng-if 指令的功能是等价的。
        <div *ngIf="condition">...</div>

ngFor Guide

        ngFor指令：使用可迭代的每个项作为模板的上下文来重复模板，类似于 Ng 1.x 中的 ng-repeat 指令
        <div *ngFor="let person of people;index as i">{{'{'}}{{'{'}}person.name{{'}'}}{{'}'}}{{'{'}}{{'{'}}i{{'}'}}{{'}'}}</div>

ngSwitch Guide

        ngSwitch指令：它包括两个指令，一个属性指令和一个结构指令。它类似于 JavaScript 中的 switch 语句

        <div [ngSwitch]='skills[0]'>
          <div *ngSwitchCase="'a'" class="alert alert-success">
            {{'{'}}{{'{'}}skills{{'}'}}{{'}'}}
          </div>
          <div *ngSwitchCase="'b'" class="alert alert-info">
            {{'{'}}{{'{'}}skills{{'}'}}{{'}'}}
          </div>
          <div *ngSwitchDefault class='alert alert-info'>
            {{'{'}}{{'{'}}skills{{'}'}}{{'}'}}
          </div>
        </div>

Section 自定义属性指令

自定义属性指令 Guide

        - 创建共同指令模块
        ng g m directives
        - 创建指令
        ng g d directives/highLight --spec false--export true
        - 指令中依赖 ElementRef HostListener Input模块,用于获取节点\绑定事件\获取绑定输入
        import {{'{'}} Directive, ElementRef, HostListener, Input {{'}'}} from '@angular/core';
